HTML helpers

  • Creating a link:
    • <a href='https://google.com'>My Cool Text</a>
    • Everything in green is the HTML code. "https://google.com" is the url your link will point towards. "My Cool Text" is what will be displayed.
    • Note the two quotation marks wrapping the url. Missing one will break the code.
    • When rendered, the link will look and operate like this: My Cool Text
  • Making text bold:
    • <strong>My Bold Text</strong>
    • Similarly, everything in green is HTML, text will appear bold when wrapping it in "strong tags."
    • When rendered, the code above will look like this: My Bold Text
  • Italicizing text:
    • <em>My Italicized Text</em>
    • When rendered, the code above will look like this: My Italicized Text
